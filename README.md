# 等保 2.0 记忆系统

一个用于记忆《网络安全等级保护基本要求》通用要求、拓展要求控制点的纯前端训练工具。

## 功能

- 按材料、分册/扩展、层面筛选控制点
- 背诵、默写、模拟三种训练模式
- 默写后进行关键词覆盖评分
- 支持随机抽题和薄弱题筛选
- 练习记录保存在浏览器本地 `localStorage`

## 数据保存说明

本项目不会连接服务器，也不会上传练习记录。每个使用者的练习次数、历史最佳分数等数据都只保存在自己的浏览器本地。

如果更换电脑、浏览器，或清理浏览器站点数据，本地记录会消失。

## 使用

直接打开 `index.html` 即可使用。部署到 GitHub Pages 后，访问 Pages 地址即可在线使用。
